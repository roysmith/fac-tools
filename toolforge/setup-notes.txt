On bastion:
* Make sure you have a deploy key installed on github
* Copy sample-user-config.py to $HOME/user-config.py
* Make user-config.py mode 0600
* Update user-config.py with appropriate OAUTH credentials

$ become fac-summary-bot
$ git clone git@github.com:roysmith/fac-tools.git

On krb backend
$ cd $HOME
$ python3 -m venv --prompt=fac-summary-bot venv
$ . venv/bin/activate
$ cd fac-tools
$ pip install .

Commands:
* toolforge webservice python3.11 shell
* toolforge jobs run bot --image python3.11 --command fac-tools/toolforge/bot.sh

To set up cron job:
* toolforge jobs run fac-summary-bot --command $HOME/fac-tools/toolforge/bot.sh --image python3.11  --schedule "*/15 * * * *" --timeout 300
